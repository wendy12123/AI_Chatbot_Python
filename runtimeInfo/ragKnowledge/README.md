# RAG Knowledge Folder

This folder stores files used by the chat retrieval flow in `ChatHandler`.

## What gets indexed
- `.md`
- `.txt`
- `.yaml` / `.yml`

## How to add content
1. Create or copy learning notes into this folder.
2. Keep text clear and topic-based so keyword matching works better.
3. Restart the chatbot runtime to rebuild the in-memory index.

## Tips
- Split very long notes into multiple files by topic.
- Include important keywords students are likely to ask about.
- Avoid putting secrets or private data here.
